# Icons Folder

This folder contains the extension icons. Please replace the placeholder instructions below with actual PNG image files.

## Required Icons

- **icon16.png** - 16x16 pixels (for browser toolbar)
- **icon48.png** - 48x48 pixels (for extension management page)
- **icon128.png** - 128x128 pixels (for Chrome Web Store)

## Creating Icons

You can create these icons using:
1. Online tools like [favicon.io](https://favicon.io) or [Canva](https://www.canva.com)
2. Image editing software like Photoshop, GIMP, or Figma
3. AI image generators like DALL-E or Midjourney

## Icon Design Suggestions

- Use the DeepSeek brand colors (purple/blue gradient)
- Include an export/download icon
- Keep it simple and recognizable
- Use transparent backgrounds for PNGs

## Temporary Solution

Until you add actual PNG files, the extension will show the default Chrome puzzle piece icon.
